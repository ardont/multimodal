# 🎙 Мультимодальный анализ речи (Газпромбанк)

MVP-решение для распознавания измененных и аномальных эмоциональных состояний человека (стресс, волнение, агрессия) по голосу и тексту.

---

## ⚙️ Особенности архитектуры

Система объединяет две модальности:
1. **Акустическую (Аудио):** извлечение акустических признаков (громкость RMS, паузы/доля тишины, темп речи BPM) + классификация на базе **Wav2Vec2** / **Hubert**.
2. **Семантическую (Текст):** автоматическое распознавание речи (ASR) через **Whisper** + классификация эмоций текста через **RuBERT**.

Для интеграции результатов используется **Late Fusion** (позднее взвешенное слияние скоров: 60% аудио, 40% текст).

### 💻 Режимы запуска (для слабых ПК и удаленных серверов)

Чтобы вы могли разрабатывать и демонстрировать проект со слабого ноутбука, в коде реализован **`MOCK_MODE`**:
* **`MOCK_MODE=True` (по умолчанию):** Запускает легкую версию. Не требует скачивания нейросетей на гигабайты и наличия GPU. Распознавание текста имитируется реалистичными сценариями, но **акустические метрики (длительность, громкость, тишина, темп BPM) извлекаются по-настоящему** с помощью библиотеки `librosa`. Это позволяет тестировать приложение и демонстрировать интерфейс на любом компьютере.
* **`MOCK_MODE=False`:** Запускает полноценный инференс нейросетей на GPU (Whisper + RuBERT + Wav2Vec2). Рекомендуется запускать на мощном удаленном сервере.

## 🚀 Быстрый старт (Локально в Mock-режиме)

### 1. Установка окружения
Убедитесь, что у вас установлен Python 3.8+ и выполните установку библиотек:
```bash
pip install -r requirements.txt
```

### 2. Запуск приложения
```bash
python app.py
```
После запуска откройте в браузере адрес [http://localhost:7860](http://localhost:7860).

## 🌐 Настройка удаленной работы через VPN (Слабый ноутбук -> Мощный ПК)

Если вы хотите запускать нейросети на мощном ПК с GPU, а управлять интерфейсом со своего ноутбука:

### Вариант А: Доступ к веб-интерфейсу напрямую через VPN
1. Объедините ноутбук и мощный ПК в одну VPN-сеть (например, через **Tailscale** или **ZeroTier** — это настраивается за 3 минуты и не требует белых IP-адресов).
2. На мощном ПК установите `MOCK_MODE=False` в окружении и запустите `app.py`:
   ```bash
   # Windows PowerShell
   $env:MOCK_MODE="False"
   python app.py
   ```
3. На слабом ноутбуке откройте браузер и введите IP-адрес мощного ПК из VPN-сети:
   `http://<IP_мощного_ПК_в_VPN>:7860`
4. Вы сможете загружать файлы и записывать голос со своего микрофона, а вычисления будут производиться на GPU мощного ПК!

### Вариант Б: Публичная ссылка Gradio
1. На мощном ПК в файле `app.py` измените параметры запуска в конце файла:
   ```python
   demo.launch(server_name="0.0.0.0", server_port=7860, share=True)
   ```
2. Запустите `app.py`. В терминале появится публичная ссылка вида `https://xxxx.gradio.live`.
3. Откройте эту ссылку на ноутбуке или телефоне.
